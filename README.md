# UniverComos

Trang web tĩnh đơn giản được publish bằng GitHub Pages từ nhánh `main`.

## GitHub Pages

Repository đã được cấu hình workflow để deploy tự động khi có thay đổi trên nhánh `main`.

Nếu đây là lần đầu bật GitHub Pages cho repository, vào:

`Settings` → `Pages` → `Build and deployment` → `Source: GitHub Actions`

Sau khi bật xong, mỗi lần push lên `main` trang sẽ được cập nhật tự động.
